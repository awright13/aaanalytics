The Bernoulli Distribution


$$ \text{Parameters:} \quad 0 < p < 1, p \in \mathbb{R} \quad \text{type: probability}\\ $$ $$ \text{Support: }x \in {0,1} \\$$ $$ \text{pmf: } p_X(x)= \begin{cases} q = (1-p) & \text{for } x=0 \\ p & \text{for } x=11 \end{cases} \\ $$ $$ \text{cdf: } F_X(x) = \begin{cases} 0 & \text{for } x<0 \\ 1 - p & \text{for } 0 \leq x < 1 \\ 1 & \text{for } x \geq 1 \\ \end{cases} \\$$ $$ \text{Mean: } E[X] = p \\ $$ $$ \text{Variance: } Var[X] = p(1-p)=pq \\ $$

A random variable $X$ that follows a Bernoulli distribution can take on one of two values $\left \{ 0,1 \right \}$. The distribution is defined by a single parameter $p$, which represents the probability that the $X$ will take on the value 1, $p = P(X=1)$. It is a special case of the more general Binomial Family, in which case the number of trials $n=1$

The Bernoulli distribution is indispensable for its ability to model events that have a binary outcome, for example success/failure, presence/absence, and belonging/not belonging (i.e. classification). While it is rare to be interested in a single event in isolation (i.e. a single r.v. X with a Bernoulli distribution), it is very common to be interested in some process or set of events, each of which follows a Bernoulli distribution. In these contexts, the Bernoulli distribution is the fundamental building block of several other interesting distributions, including:

The Binomial Distribution: distribution of the number of successes out of $n$ i.i.d. Bernoulli trials, each with probability of success $p$
The Geometric Distribution: distribution of the number of i.i.d. Bernoulli trials required to obtain one success
The Negative Binomial Distribution: distribution of the number i.i.d. Bernoulli trials required to obtain the $r$-th success
The Poisson Distribution: As $n \rightarrow \infty$ and $p \rightarrow 0$, the binomial distribution converges to Poisson as the number of occurences of i.i.d. Bernoulli events in a given system, in a given time/space frame, at a given mean rate $\lambda$

The Beta distribution is an essential component in Bayesian inference. It serves as the conjugate prior probability distribution for the following distributions:

Bernoulli
Binomial
Negative Binomial
Geometric

The primary difference between Bayesian statistics and Frequentist statistic is that Frequentist statistics assumes that the parameters of a given distribution have fixed and definite (albeit unknown) values, whereas in Bayesian statistics we treat parameters as having a probability distribution, through which we can make statements of belief/expectation/confidence in regards to the value of the parameter. That’s where the Beta distribution comes in to play – it is used to model the probability distribution of the probability of success for the underlying process.


The pdf of the Beta distribution has two parameters, $\alpha$ and $\beta$. We can interpret these parameters as the accumulation of cases that fall into the two values of a binary classification problem. Typically, $\alpha$ is designated as 'the number of successes’ while $\beta$ is ‘the number of failures’.

In Bayesian inference, when initiating a model using the Beta distribution, one can use their ‘subjective belief’ about the subject to select the values of $\alpha$ and $\beta$. Then, as classifications are generated according the underlying process (Bernoulli, Binomial, Neg Bin, Geom), the parameters $\alpha$ and $\beta$ are updated ($\alpha$ + 1) for success, ($\beta$ + 1) for failure. Once the parameters have been updated, we now have our revised Beta distribution for the parameter of the underlying process ($p$=probability of success).
